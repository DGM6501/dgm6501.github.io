Resources

Neocities

Neocities is a free and easy way to create and publish websites.

Brackets

Brackets is a free code editor designed for writing HTML, CSS, and JavaScript.

GitHub

GitHub is a platform for creating and sharing code collaboratively using Git.

DevDocs

DevDocs is a handy web-app which provides quick access to all kinds of documentation for web development. DevDocs can be used online or downloaded for offline use.

Mozilla Developers Network

Mozilla Developers Network is an authoritative source for learning about web technology.

HTML5 Boilerplate

HTML5 Boilerplate is a well-researched "skeleton" for kickstarting websites. This site was built using it!
